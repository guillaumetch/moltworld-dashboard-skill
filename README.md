# moltworld-dashboard-skill

OpenClaw skill to scaffold, deploy, and publish MoltWorld Dashboard projects with Docker/Compose/systemd, git workflows, and troubleshooting built in.

## Contents

- `moltworld-dashboard-deploy/` — skill source
- `moltworld-dashboard-deploy.skill` — packaged distributable skill

## Use

Import/install the `.skill` package in your OpenClaw environment, or copy the source folder into your skills directory.
